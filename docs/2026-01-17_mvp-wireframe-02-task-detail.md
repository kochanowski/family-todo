## Wireframe

```
╔══════════════════════════════════════════════════╗
║ < My Tasks                               [Edit] ║
╠══════════════════════════════════════════════════╣
║  Buy milk                                       ║
║                                                ║
║  ┌───────────────────────────────────────────┐ ║
║  │ Assigned to: You                          │ ║
║  │ Area: Kitchen                             │ ║
║  │ Due: Today                                │ ║
║  │ Status: Next                              │ ║
║  │ Recurring: Off                            │ ║
║  └───────────────────────────────────────────┘ ║
║                                                ║
║  Notes                                         ║
║  ┌───────────────────────────────────────────┐ ║
║  │ 2L milk, check eggs                        │ ║
║  └───────────────────────────────────────────┘ ║
║                                                ║
║  History                                       ║
║  ┌───────────────────────────────────────────┐ ║
║  │ Created by You · Today 09:15               │ ║
║  │ Moved to Next · Today 09:20                │ ║
║  └───────────────────────────────────────────┘ ║
║                                                ║
║                [Mark as Done]                  ║
╚══════════════════════════════════════════════════╝
```

## Interakcje
- Tap `[Edit]` → edycja zadania
- Tap `[Mark as Done]` → oznaczenie jako ukończone
- Tap `< My Tasks` → powrót do listy
